Obtaining an API Key

Getting an API key is the first step to interacting with the EIA database. You will include this key in the URL query string when making requests to the site. To get your key, first you need to go register at the EIA website. The purpose of this registering and getting the key is so that the EIA can notify you of any changes to the API. They don't state it on their website, but they might also use it to throttle your usage in case you are making too many requests in a short period of time.
